# Data Analytics Overview

## What I Learned
Today I reviewed a peer coaching session from the GROW module.

### The 5 Vs of Data
- Volume – amount of data
- Velocity – speed of data generation
- Variety – different types of data
- Veracity – data quality
- Value – usefulness of data

### What is Data Analytics?
Data analytics is the process of collecting, cleaning, analyzing, and interpreting data to make decisions.

### Processes of Data Analytics
1. Defining your objective
2.  Data collection
3. Data cleaning
4. Data analysis
5. Data sharing (reports, dashboards)

### Types of Data Analysis
- Descriptive Analysis
- Diagnostic Analysis
- Predictive Analysis
- Prescriptive Analysis

### Types of Analytical Methods
- Univariate Analysis
- Bivariate Analysis
- Time Series Analysis
- Regression Analysis

### Tools Mentioned
- Data Management: Salesforce, SAS, Xplenty
- Visualization: Tableau, Datawrapper
- Communication: Google Charts

## My Understanding
Data analytics is a full cycle, not just analysis. Cleaning and communication are just as important.
